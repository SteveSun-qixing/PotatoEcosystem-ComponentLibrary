# 功能需求：主题系统与分层回退

## 1. 主题目标

支持多主题深度定制，覆盖颜色、字体、圆角、阴影、体积层级、动画参数、密度、边框策略。

## 2. 分层作用域

主题按以下层级生效：

1. 全局层（System）
2. 应用层（App）
3. 窗口层（Window/Container）
4. 组件层（Instance）

## 3. 回退机制

- 当前层未配置主题时，自动向父层查找。
- 若逐级均为空，回退到系统默认主题。
- 回退机制必须对颜色、动效、体积、排版等全部 token 生效，不限于配色。

## 4. 主题包约束

- 主题包不可修改组件结构与业务逻辑。
- 主题包必须遵循统一 token 命名契约。
- 主题包应支持按组件按需加载，避免一次性全量注入。
