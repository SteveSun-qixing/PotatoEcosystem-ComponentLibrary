# 验收标准与里程碑出口

## 1. 里程碑

- M1：仓库重置完成，工程基线稳定。
- M2：token schema 与主题流水线完成。
- M3：P0 组件完成并通过测试门禁。
- M4：主题分层回退能力完成。
- M5：Core / Foundation / SDK 联调完成。
- M6：首版发布候选（RC）完成。

## 2. 验收维度

1. 架构验收：是否完全符合“逻辑与样式解耦”。
2. 规范验收：是否符合生态开发规范总则。
3. 功能验收：P0 组件是否达到交互与可访问性标准。
4. 集成验收：是否与 SDK / Foundation 对接稳定。
5. 文档验收：需求、技术、计划是否可追踪到代码实现。

## 3. 出口条件

- 所有阻断缺陷清零。
- 质量门禁全绿。
- 发布清单、回滚预案、版本说明齐全。
