# 总体技术架构

## 1. 分层架构

```text
Chips Apps / Plugins
        |
Chips Component Library (Headless)
        |
Chips SDK
        |
Chips Foundation
        |
Chips Core (Router)
```

## 2. 组件库内部架构

```text
packages/
  tokens/      # 设计令牌源
  themes/      # 主题包产物与映射
  primitives/  # Ark UI 二次封装
  components/  # 业务语义组件
  adapters/    # SDK/Foundation 适配层
  tooling/     # 构建、测试、发布工具
```

## 3. 核心原则

- 协议优先：先冻结 token 和接口契约，再进入组件编码。
- 运行时最小化：主题切换尽量依赖 CSS 变量继承而非 JS 频繁重算。
- 可扩展：支持多主题、多语言、多终端渲染。
