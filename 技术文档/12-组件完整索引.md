# 组件完整索引

## 组件分类概览

Chips前端组件库（Vue 3）包含通用UI组件，分为以下几大类：

| 分类 | 数量 | 说明 |
|------|------|------|
| 基础组件 | 25个 | 最基本的交互元素 |
| 布局组件 | 13个 | 页面结构和空间组织（含 BoxLayoutView） |
| 表单组件 | 18个 | 表单输入和验证 |
| 数据展示组件 | 15个 | 数据展示和可视化 |
| 反馈组件 | 10个 | 用户反馈和提示 |
| 导航组件 | 8个 | 页面导航和路由 |
| 卡片辅助组件 | 5个 | 卡片开发辅助工具 |
| 特殊组件 | 8个 | 系统级组件 |

**总计**: 约100个组件

> **重要说明**：具体的基础卡片组件（如 MarkdownCard、VideoCard、ImageCard 等）**不属于组件库**，它们应该在各自的**基础卡片插件**中实现。组件库只提供卡片辅助组件供插件开发时使用。

### 1. Button 按钮

**文件位置**: `packages/core/src/components/Button/`

**Props接口**:
```typescript
interface ButtonProps {
  type?: 'default' | 'primary' | 'dashed' | 'link' | 'text';
  size?: 'small' | 'medium' | 'large';
  disabled?: boolean;
  loading?: boolean;
  danger?: boolean;
  icon?: ReactNode;
  htmlType?: 'button' | 'submit' | 'reset';
  children?: ReactNode;
  onClick?: (event: MouseEvent<HTMLButtonElement>) => void;
  className?: string;
  style?: CSSProperties;
}
```

**事件列表**:
- `onClick`: 点击事件
- `onMouseEnter`: 鼠标进入
- `onMouseLeave`: 鼠标离开
- `onFocus`: 获得焦点
- `onBlur`: 失去焦点

**样式接口点**:
- `.chips-button`: 按钮容器
- `.chips-button-primary`: 主要按钮
- `.chips-button-small/medium/large`: 尺寸变体
- `.chips-button-disabled`: 禁用状态
- `.chips-button-loading`: 加载状态
- `.chips-button-icon`: 图标容器

**使用示例**:
```typescript
<Button type="primary" loading onClick={handleClick}>
  提交
</Button>
```

- `.chips-card-placeholder__icon`: 图标

- `.chips-card-placeholder__action`: 操作按钮

## 组件注册和使用

```typescript
import { registerComponent } from '@chips-ui/core';
import { MyCustomComponent } from './MyCustomComponent';

registerComponent('MyCustomComponent', MyCustomComponent);
```

### 批量导入

```typescript
// 导入所有组件
import * as ChipsUI from '@chips-ui/react';

// 按需导入
import { Button, Input, Select } from '@chips-ui/react';
```

### 树摇优化

组件库支持Tree Shaking，只会打包使用的组件:

```typescript
// 只会打包Button组件
import { Button } from '@chips-ui/react';
```

## 总结

薯片组件库（Vue 3）提供了完整的通用 UI 组件体系，覆盖了从基础交互到复杂业务场景的各种需求。所有组件都遵循统一的设计规范，提供清晰的 Props 接口、事件系统和样式接口点，确保了易用性和可扩展性。通过主题系统，所有组件都可以实现视觉定制，满足不同项目的设计需求。

**重要说明**：
- 组件库**不包含**具体的基础卡片组件（如 MarkdownCard、VideoCard 等）
- 基础卡片组件应该在各自的**插件**中实现
- 组件库只提供**卡片辅助组件**供插件开发时使用
