## 项目概述

薯片组件库是薯片生态的官方UI组件库，为开发者提供构建薯片应用前端界面的基础组件。组件库遵循**功能与样式完全分离**的设计原则，组件只实现功能逻辑和布局结构，不包含任何样式代码，所有视觉效果由主题包提供。

### 核心特点

- **Vue 3 + TypeScript**: 使用 Composition API 和 TypeScript 5.0+
- **无样式组件**: 组件不包含任何CSS样式代码
- **主题系统支撑**: 暴露CSS类名作为样式接口点
- **多语言支持**: 所有文本使用系统词汇编码
- **中心路由通信**: 通过微内核调用公共基础层功能
- **完整类型定义**: 提供完整的TypeScript类型

## 开发原则

### 必须遵守的规范

1. **功能与样式分离**: 组件代码不包含任何样式（颜色、字体、间距等）
2. **多语言支持**: 不硬编码任何文本，使用 `t()` 函数
3. **中心路由**: 所有对基础层的调用通过 `Core.request()` 
4. **样式接口点**: 使用 `chips-` 前缀的CSS类名
5. **TypeScript**: 完整的类型定义
6. **测试覆盖**: 核心组件测试覆盖率 ≥80%

### 代码规范

- 使用 Vue 3 Composition API
- 使用 `<script setup lang="ts">` 语法
- 组件文件使用 PascalCase 命名
- 工具函数使用 camelCase 命名
- 遵循 ESLint 和 Prettier 配置

### 提交规范

- 每完成一个任务提交一次
- 使用规范的 commit message 格式
- type(scope): subject
