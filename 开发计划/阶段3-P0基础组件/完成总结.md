## 组件功能详情

### Button 按钮
- 5种类型: default, primary, dashed, link, text
- 3种尺寸: small, medium, large
- 支持禁用、加载、危险、块级状态
- 支持图标插槽

### Input 输入框
- 6种输入类型: text, password, number, email, tel, url
- 支持前缀/后缀插槽
- 支持清除按钮和字数统计
- 支持受控/非受控模式

### Textarea 多行文本框
- 支持自动高度调整（autoSize）
- 支持字数统计和最大长度
- 支持 resize 属性

### Select 选择器
- 支持单选和多选
- 支持搜索过滤
- 支持键盘导航（↑↓ Enter Escape）
- 支持清除和禁用选项

### Checkbox / CheckboxGroup
- 支持半选状态（indeterminate）
- 支持 options 配置和插槽嵌套
- provide/inject 组件联动

### Radio / RadioGroup
- 支持按钮样式单选
- 支持 options 配置和插槽嵌套
- 支持水平/垂直排列

### Switch 开关
- 支持自定义开关内容（checked/unchecked 插槽）
- 支持加载状态
- 支持自定义值

## 样式接口点汇总

所有组件使用 `chips-` 前缀的 CSS 类名作为样式接口点：

```
Button:     .chips-button, .chips-button--primary, etc.
Input:      .chips-input, .chips-input__inner, etc.
Textarea:   .chips-textarea, .chips-textarea__inner, etc.
Select:     .chips-select, .chips-select__dropdown, etc.
Checkbox:   .chips-checkbox, .chips-checkbox--checked, etc.
Radio:      .chips-radio, .chips-radio-group, etc.
Switch:     .chips-switch, .chips-switch__handle, etc.
```

## 开发方法

本阶段使用了 **并行子代理** 开发模式：

- 4个子代理同时开发 Input、Select、Checkbox、Radio/Switch
- 大幅提高开发效率
- 每个子代理独立完成组件实现和测试
