## 项目概述

薯片组件库是薯片生态的官方UI组件库，采用 **无样式组件** 设计理念，所有视觉样式由独立的主题包提供。组件库提供完整的语义化 CSS 类名作为样式接口点，实现了与主题系统的完美解耦。

## 核心特性

### 1. 无样式设计

所有组件不包含任何 CSS 代码，仅提供语义化的 CSS 类名作为样式接口点。主题系统通过这些类名实现视觉定制。

### 2. ARIA 无障碍

所有交互组件都实现了完整的 ARIA 属性，支持:
- 屏幕阅读器
- 键盘导航
- 焦点管理

### 3. 受控/非受控模式

表单组件同时支持:
- **受控模式**: 通过 `v-model` 绑定
- **非受控模式**: 通过 `defaultValue` 初始化

### 4. TypeScript 支持

完整的类型定义，包括:
- Props 接口
- Emits 定义
- 暴露方法类型
- 插槽类型

### 5. 生态集成

- **主题上下文**: 与 Theme 系统集成
- **内核通信**: 通过 Chips-Core 路由
- **多语言支持**: i18n 系统集成
